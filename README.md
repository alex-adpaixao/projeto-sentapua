# Projeto SentaPua – Maratona de Programação

Este repositório tem como objetivo apoiar a formação de estudantes em Programação Competitiva, preparando alunos do ensino médio e superior para competições como a OBI (Olimpíada Brasileira de Informática) e a Maratona de Programação da SBC.

O projeto reúne materiais de estudo, exemplos de código, listas de exercícios e desafios progressivos, abordando desde lógica de programação até estruturas de dados e algoritmos utilizados em competições.

A proposta é desenvolver o raciocínio lógico, a capacidade de resolução de problemas e o domínio de técnicas de programação utilizando principalmente a linguagem C/C++.

O conteúdo está organizado em módulos semanais, permitindo que os estudantes acompanhem a evolução do aprendizado por meio de teoria, prática e resolução de problemas em juízes online como Beecrowd, Codeforces e CSES.

Projeto desenvolvido no âmbito do Instituto Federal de Sergipe (IFS), com foco na preparação de equipes e no incentivo ao estudo de algoritmos e estruturas de dados.
